# marathon-decomp

[![DOI](https://zenodo.org/badge/1279629652.svg)](https://doi.org/10.5281/zenodo.20999281)

Code and a **synthetic dataset** reproducing the analysis in:

> **Ranking Marathon Difficulty from Large, Sparse Data: A Robust Latent-Factor Approach**
> Yi Hua (Dillon) Chang (2026).

This repository decomposes marathon finish times into separable *athlete* and
*race* latent factors to rank race difficulty independently of who showed up.

## Data is synthetic

`data/race_results/` is a **synthetic twin** of the private source data: finish
times carry multiplicative jitter, athlete birth-year bounds are perturbed,
country labels are anonymized to opaque codes, and the source-database athlete
linkage is removed. Race-level metadata (series, date, year) and the per-race
**covariate tables** (weather, course elevation) are real public-event facts.
Results reproduced from this data are *qualitatively* similar but not identical
to the paper. See `data/race_results/manifest.json` for the exact perturbation.

The synthetic export is regenerated by
`scripts/01_data_prep/make_fake_export.py` from the (unpublished) real export.

## Install

```bash
pip install -e .
# optional: also installs statsmodels, needed only for the random-course
# mixed-model covariate cross-check (scripts/05_analysis/covariate/q06_*).
pip install -e .[reports]
```

## Run

The analysis pipeline is staged under `scripts/01..06` (each stage is
self-documenting). A typical flow: fit the models (`scripts/03_model_fit/`),
then run the analyses (`scripts/05_analysis/`). Pre-computed headline result
tables ship under `results/analysis/`.

## Layout

```
src/marathon_decomp/   modeling package (model, aging basis, data loader)
scripts/01_data_prep/  synthetic-data generation + dataset summary
scripts/02..06/        model selection, fits, validation, analysis, sensitivity
data/race_results/     synthetic athlete x race export
data/weather/, data/course_profile/, data/misc/   real per-race covariates
results/analysis/      pre-computed headline tables (race rankings, etc.)
```
